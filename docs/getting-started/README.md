# Getting Started

Welcome to FlowGuard! This section contains everything you need to install, configure, and start using FlowGuard effectively.

## Guides in This Section

1. [Installation Guide](installation.md) - How to install and set up FlowGuard
2. [First Epic](first-epic.md) - Creating your first epic
3. [Configuration](configuration.md) - Setting up your preferences
4. [Quick Reference](quick-reference.md) - Essential shortcuts and commands

## Recommended Learning Path

If you're new to FlowGuard, we recommend following this order:

1. Start with the [Installation Guide](installation.md) to get FlowGuard running in your environment
2. Follow the [First Epic](first-epic.md) guide to understand the core concepts
3. Review the [Configuration](configuration.md) guide to customize FlowGuard to your needs
4. Keep the [Quick Reference](quick-reference.md) handy for daily usage

## Interactive Learning

For a hands-on introduction, try starting the [First Epic Tutorial](../tutorials/first-epic-tutorial.md) from the Command Palette using "FlowGuard: Start Tutorial".

## Need Help?

If you encounter any issues during setup, check our [Troubleshooting Guide](../troubleshooting.md) or visit the [FAQ](../faq.md) for common questions.
