# Interactive Tutorials

FlowGuard includes interactive tutorials to help you learn the system through guided walkthroughs directly in VS Code.

## Available Tutorials

### First Epic Tutorial
Learn how to create your first epic, specification, and ticket in FlowGuard. This tutorial guides you through:
- Initializing a new epic
- Creating a specification
- Editing specification metadata
- Generating tickets from specifications
- Creating agent handoffs

To start this tutorial, run the `FlowGuard: Start First Epic Tutorial` command from the command palette.

### Handoff Tutorial
Learn how to create and use agent handoffs for AI-assisted development. This tutorial covers:
- Preparing tickets for handoff
- Generating handoff documents
- Customizing handoff content
- Using handoffs with AI assistants
- Tracking handoff execution

To start this tutorial, run the `FlowGuard: Start Handoff Tutorial` command from the command palette.

### Verification Tutorial
Learn how to verify your code changes with FlowGuard. This tutorial covers:
- Making code changes
- Running verification on your changes
- Reviewing verification results
- Applying auto-fixes
- Approving verification results

To start this tutorial, run the `FlowGuard: Start Verification Tutorial` command from the command palette.

## How Tutorials Work

Tutorials provide step-by-step guidance with:
- Clear instructions for each step
- Helpful hints and tips
- Automatic validation of completed steps
- Progress tracking
- Ability to skip or go back to previous steps

## Accessing Tutorials

You can access tutorials in two ways:

1. **Command Palette**: Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac) and search for "FlowGuard: Start Tutorial"
2. **Welcome Screen**: Tutorials are featured on the FlowGuard welcome screen when you first open the extension
