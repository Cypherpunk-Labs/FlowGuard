# Advanced Topics

This section covers advanced topics for power users and developers who want to extend FlowGuard's functionality. These guides assume familiarity with the core concepts and features.

## Topics in This Section

1. [Plugin Development](plugin-development.md) - Creating custom plugins
2. [Custom Verification Rules](custom-verification-rules.md) - Writing verification rules
3. [LLM Integration](llm-integration.md) - Integrating language models
4. [Custom Agents](custom-agents.md) - Creating agent templates
5. [Extending UI](extending-ui.md) - Customizing the user interface

## Prerequisites

These guides assume you have:
- Experience with the core FlowGuard workflows
- Familiarity with software development concepts
- Understanding of the FlowGuard architecture (see [Architecture Reference](../reference/architecture.md))
- For plugin development: Knowledge of TypeScript and VS Code extension development

## Getting Started

If you're new to FlowGuard, we recommend starting with the [Getting Started](../getting-started/) and [User Guides](../guides/) sections before exploring these advanced topics.

## Examples

Each guide includes practical examples and references to the FlowGuard codebase. You can find additional examples in the [examples directory](../examples/).

## Need Help?

If you're having trouble with any of these advanced topics, consider starting with the [Plugin Development Tutorial](../tutorials/plugin-tutorial.md) or checking the [FAQ](../faq.md).
