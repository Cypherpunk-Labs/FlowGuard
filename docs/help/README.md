## Getting Help

If you're experiencing issues with FlowGuard, we recommend:

1. First, check the [Troubleshooting Guide](../troubleshooting.md) for solutions to common problems
2. Search the [FAQ](../faq.md) for answers to frequently asked questions
3. Review the relevant documentation sections:
   - [Getting Started](../getting-started/) for basic usage
   - [User Guides](../guides/) for workflow-specific help
   - [Reference](../reference/) for technical details

## Reporting Issues

If you've encountered a bug or have a feature request:

1. Check the [existing issues](https://github.com/your-repo/issues) to see if it's already been reported
2. If not, [open a new issue](https://github.com/your-repo/issues/new) with:
   - A clear description of the problem
   - Steps to reproduce the issue
   - Your FlowGuard version and VS Code version
   - Any relevant error messages or logs
