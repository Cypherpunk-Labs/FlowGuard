# Icons

This directory contains the FlowGuard logo and icon variations.

## Available Icon Placeholders

This directory includes placeholder files for icons and logo assets:

1. `logo.svg.txt` - FlowGuard logo (SVG format)
2. `extension-icon.png.txt` - Extension icon (128x128 PNG)
3. `sidebar-icons.txt` - Sidebar icons for different views
4. `status-icons.txt` - Status icons (draft, planning, in_progress, completed)
5. `severity-icons.txt` - Severity level icons (critical, high, medium, low)
6. `file-type-icons.txt` - File type icons for specs and tickets
